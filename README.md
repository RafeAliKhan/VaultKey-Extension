# VaultKey-Extension
---

VaultKey is a privacy-first Chrome extension that stores passwords, payment cards, and secure notes entirely on your device — no cloud, no account, no subscription. Built with pure HTML, CSS, and JavaScript in under 50 KB.

---

## 📖 About

VaultKey is a lightweight, zero-dependency Chrome browser extension that gives you a secure personal vault right inside your toolbar. Everything is protected behind a master password and stored locally using Chrome's built-in Storage API — your data never touches an external server.

Built entirely with HTML5, CSS3, and Vanilla JavaScript, VaultKey is under 50 KB in total size and works completely offline.

---

## 🎨 Design

- Deep-space gradient UI with animated glow orbs
- Smooth slide-up modals and entry card animations
- Syne (display) + DM Sans (body) typefaces
- Fully responsive inside Chrome's extension popup constraints
